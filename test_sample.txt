RAG Chatbot System Test Document

This is a test document for our RAG (Retrieval-Augmented Generation) chatbot system.

The system includes:
- Rasa for conversational AI
- ChromaDB for vector storage
- Redis for caching
- PDF processing with text extraction
- Sentence-transformers for embeddings

Key Features:
1. Upload PDF documents
2. Extract and chunk text content
3. Generate embeddings using all-MiniLM-L6-v2 model
4. Store embeddings in ChromaDB
5. Answer questions based on document content

This document contains test information about our Rasa chatbot that can answer questions about PDF content.
